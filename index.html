Exploring P5.js

Welcome to my project in p5.js! I am self learning about rendering graphics and I am trying to learn the fundamentals. I am using a specific tool called p5.js. P5 is a JavaScript library that makes code more accessible and easy to interact with. Click here to go to p5's website. I also used this project as an opportunity to learn about git, JavaScript, and p5. All of the code that went into this project can be found here.

Simple Shape

I started out with drawing basic shapes to learn how p5 works -- just squares and ellipses; building blocks for more complex things. Once I figured out the draw() and setup() functions, I moved on.

The Grid

One idea was to move toward building maze-like structures, so I rendered a grid of different colored squares. In this case, we are rendering the grid from a two-dimensional array and drawing the squares randomly over the grid.

Adding Rules

I wanted to build a more complex algorithm for creating a solvable maze, but I also wanted to decide whether a specific square would be light or dark to build a path through a maze. I tried several combinations to make sure that the white squares didn't clump together and create chambers; after a few tries, I figured out this solution. However, it didn't look like a very realistic maze

The Snake

In order to make a more realistic path, I rethought the code. I eventually decided to change the perspective; instead of randomly choosing a square and deciding what color it would be, I decided the square based on the previous square. I also explored color and how each square can have its own changing color. Once I had written the rules for the "snake" and watched it, I campe up with new about the project.

Conclusion

There are many possibilites about where this project might go: I could draw a grid using input from data sets -- right now, most of the data is randomly generated -- such as the value of the dollar or the population of bengal tigers. Another idea is creating generative art such as mazes, landscape drawings, or just abstract art.
